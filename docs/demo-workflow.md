# Gyors bemutato workflow

Ez a leiras az Iteracio 5 bemutatohivo akcioterve.

1. Inditsd el az alkalmazast `cargo run`-nal vagy a release buildbol.
2. A dashboardon ellenorizd, hogy a demo adat betoltodott-e.
3. Nyisd meg a `Rust alapok` paklit.
4. Indits tanulasi sessiont, opcionalsan kapcsold be a keverest.
5. Forditsd meg a kartyat `Space`-szel.
6. Ugorj tovabb a kovetkezo kartyara a nyilakkal vagy a kovetkezo gombbal.
7. Ertekeld a kartyat `1` = Nehez, `2` = Kozepes, `3` = Konnyu.
8. Zard le a sessiont, majd ellenorizd, hogy a session osszegzes megjelenik az allapotban.
9. Mentsd el az allapotot a floppys ikonnal, majd inditsd ujra az appot, hogy latszodjon a JSON perzisztencia.
